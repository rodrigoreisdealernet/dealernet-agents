# Dealernet Agents

> Um assistente de IA que vigia a concessionária 24/7, encontra sozinho onde se está
> perdendo ou deixando de ganhar dinheiro, e entrega cada oportunidade pronta para o
> dono/gestor aprovar com um clique — sem precisar pedir relatório nenhum.

## O que é

**Dealernet Agents** é a camada **agêntica e proativa** sobre o DMS Dealernet, voltada a
**donos e managers de concessionárias**. Em vez de o usuário consultar relatórios (modelo
*pull*), uma frota de **agentes de IA** varre continuamente os dados da concessionária
(vendas, estoque, oficina/OS, financeiro, CRM), **detecta oportunidades e riscos** e
**propõe ações com evidência e nível de confiança** — entregues como uma fila de
*insights/findings* e *briefs* executivos que o humano **aprova ou rejeita**.

Princípio inegociável: **"agents propose; humans dispose"** — nenhum agente aplica ação
irreversível sem aprovação humana (human-in-the-loop).
